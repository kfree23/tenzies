# Tenzies

A dice game where you roll until all dice show the same number. Click to "freeze" dice between rolls and race to match them all!

## How It Works
- Roll all 10 dice
- Click a die to freeze it at its current value
- Keep rolling unfrozen dice until all 10 match
- Win and celebrate! 🎉

## Features
- Interactive dice with freeze/unfreeze toggle
- Roll counter or timer (track your best game)
- Win detection with celebration state
- Clean, responsive UI

## Tech Stack
- React
- JavaScript
- CSS
